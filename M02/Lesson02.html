Creating An Array

From a list or an iterable object:

a = np.array([1,2,3,4,5])

A 2 dimensional array with explicit data type:

a = np.array([[1,2],[3,4],[5,6]], dtype=np.float32)

A 2 x 3 array filled with zeros:

np.zeros((2,3))

A 3 x 2 array filled with ones:

np.ones((3,2))

Fill with explicit shape, value and type:

np.full((3,2), 123, dtype=np.float64)

Fill incrementally between 0.0 (inclusive) and 10.0 (exclusive) by 1 :

np.arange(0.0, 10.0, 1.0)

Create a 21 element array with linearly spaced values between 0 and 10 inclusively:

np.linspace(0.0, 10.0, 21)

Create 10 random values between -1 and 1 :

np.random.uniform(-1, 1, size=10)

Derive 20 random values from a normal distribution that is centered at 10 and has a standard deviation of

mu = 10 sigma = 0.1 np.random.normal(mu, sigma, size=20)

Indexing and Slicing

Example Description
a[i] Select element at index i , where i is an integer
a[-i] Select element at index i from the end of the array, where -1 is the last element and i is an integer
a[:] Select all the elements in the array
a[i:] Select all elements starting from index i
a[:i] Select all elements between index 0 (inclusive) and i (exclusive)
a[i:j] Select all elements between index i (inclusive) and j (exclusive)
a[i:j:k] Select all elements between index i (inclusive) and j (exclusive) with increment k
a[::-1] Select all elements in reserve order

Note: Slices are views into the array. A modification of the view modifies the array.

a = np.array([[1,2],[3,4],[5,6]], dtype=np.float32) a

produces:

array([[ 1., 2.], [ 3., 4.], [ 5., 6.]], dtype=float32)

b = a[1] b[:] = [30,40] a

produces:

array([[ 1., 2.], [ 30., 40.], [ 5., 6.]], dtype=float32)

Make sure to use .copy() to have an explicit copy rather than a view into the original array.

a = np.array([[1,2],[3,4],[5,6]], dtype=np.float32) a

produces:

array([[ 1., 2.], [ 3., 4.], [ 5., 6.]], dtype=float32)

b = a[1].copy() b[:] = [30,40] a

produces:

array([[ 1., 2.], [ 3., 4.], [ 5., 6.]], dtype=float32)

One of my favorite slicing techniques is to use boolean-valued indexing.

a = np.linspace(0.0, 10.0, 21) a[a>5.0]

produces:

array([ 5.5, 6. , 6.5, 7. , 7.5, 8. , 8.5, 9. , 9.5, 10. ])

Reshaping

Remember, the data in a NumPy array is one long sequence of elements, its shape interprets it.

np.arange(0,10,1).reshape((5,2))

In the above example, we created an array of 10 elements and reshape it to 5 rows with 2 columns. BTW, row, column is the default interpretation.

There exist 2 functions ( flatten , ravel ) to return a single dimension array of the data. The function ravel returns a view of the data, where the function flatten returns a copy of the data.

np.array([ [1,2], [3,4], [5,6] ]).flatten()

produces:

array([1, 2, 3, 4, 5, 6])

Arrays can be stacked vertically and/or horizontally:

a = np.array([1,2,3]) b = np.array([4,5,6])

np.vstack((a,b))

produces:

array([[1, 2, 3], [4, 5, 6]])

np.hstack((a,b))

produces:

array([1, 2, 3, 4, 5, 6])

Numerical Processing

The prefix num in NumPy is an indication to perform numerical processing on the data elements. A way to perform numerical efficiency is through native vectorization, where the need to define an explicit for-loop on each element is eliminated.

Numerical processing/vectorization can be arithmetic between NumPy arrays and scalar values, for example:

a = np.array([1,2,3]) b = np.array([4,5,6])

a + b

produces:

array([5, 7, 9])

a * b

produces:

array([ 4, 10, 18])

a + 10

produces:

array([11, 12, 13])

a * 3

produces:

array([3, 6, 9])

In addition, vectorization can be performed element wise, such as in the case of the application of mathematical functions. For example:

x = np.linspace(-1,1,11) np.cos(np.pi*x).round(6)

produces:

array([-1. , -0.809017, -0.309017, 0.309017, 0.809017, 1.

0.809017, 0.309017, -0.309017, -0.809017, -1.

,

])

Aggregation is yet another powerful set of functions that can be applied to NumPy arrays, such as np.mean ,

np.std , np.min and np.argmin . For example:

x = np.random.uniform(-180, 180, size=100) y = np.random.uniform(-90, 90, size=100) xy = np.stack((x, y),axis=1) xy.mean(axis=0)

produce: (results may vary)

array([ 7.07869795, -2.36257256])

In the above example, the vectorized result is the average values of the first and second columns in the xy array.

Something to remember is that all the elements in an array must have the same data type.

Using Seaborn

“If matplotlib tries to make easy things easy and hard things possible”, Seaborn tries to make a well-defined set of hard things easy too.”

Seaborn is build on top of matplotlib, but comes with a set of features to make things easier and more visually expressive.

The following is a “modern” representation of the above histogram:

import seaborn as sns

# set default aesthetic parameters

sns.set() ax = sns.countplot(hour_of_day)

And as mentioned in the above statement, “hard things become easy too”. The following is a kernel density estimation of pickup locations in Manhattan, NY. The data is downloaded from New York City Taxi Trip Duration Kaggle competition.

# Set default aesthetic parameters

sns.set()

# Set spatial boundary for lower Manhattan

lon_min, lat_min, lon_max, lat_max = [-74.048369, 40.698025, -73.907752, 40.782204]

# Load the training data and extract only lon/lat values

data = np.loadtxt('train.csv', skiprows=1, delimiter=',', usecols=(5,6))

# Make views

x = data[:,0] y = data[:,1]

# Boolean filter the data based on spatial boundary

data = data[(lat_min<y) & (y<lat_max) & (lon_min<x) & (x<lon_max)]

# Create a KDE plot of the first 100,000 records

n = 100000 ax = sns.kdeplot(data=data[:n,0],data2=data[:n,1],shade=True)

The shape of Manhattan is recognizable, and there seem to be a lot of pickups near the Lincoln tunnel on the west side.
